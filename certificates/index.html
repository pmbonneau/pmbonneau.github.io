Updated SSL Certificates for Let's Encrypt

Some old web browsers don't have updated Root certificate for sites that use Let's Encrypt SSL. So, those browsers will most likely return security errors or not work at all when visiting websites that are signed with newer certificates.


A potential solution for this issue is to manually install newer Let's Encrypt Root certificates.
